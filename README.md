# learning_js

learning_js is a server that receives GET/POST requests, processes information and sends back data in JSON format.

Technologies used
------------------------------
* Back-end
     - Node.js
     - Express framework
     - MongoDB (mongoose)
* Front-end
     - React
     - Bootstrap template (https://bootswatch.com/flatly/)
     - JQuery
